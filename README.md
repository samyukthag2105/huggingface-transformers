# Hugging Face Transformers

This project contains beginner-friendly Natural Language Processing (NLP) applications built using the Hugging Face Transformers library.

## Features

- Sentiment Analysis
- Text Summarization
- AI Title Generation

## Technologies Used

- Python
- Hugging Face Transformers
- PyTorch

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
python summarizer.py
```
